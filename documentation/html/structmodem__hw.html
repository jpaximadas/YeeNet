Detailed Description

modem_hl.h

Define high-level functions for interacting with SX127x chip. Alternate platforms must implement:

  • modem_transmit()
  • modem_listen()
  • modem_load_payload()
  • modem_get_payload()
  • modem_setup()
  • rand_32()
